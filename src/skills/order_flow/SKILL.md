# 文字下单流程（流程 B）

## 步骤

B1. **提取参数** - LLM 从用户输入中提取客户名、商品名、数量、单位、颜色、仓库
B2. **查询信息** - 搜索客户（CompanyList）+ 搜索商品（ProductList），一件完成
B3. **查价格** - 优先客户历史成交价，其次零售价
B4. **库存决策** - 非礼盒类跳过，礼盒类查库存并可能询问用户
B5. **开销售单** - SalesAdd，所有商品一次性开单

## 库存决策规则

### 不查库存的类别（直接从百鑫发货）
泡袋/包茶/内衬/PVC/标签/纸箱/烫金泡袋/提袋UV/提袋丝印/烫膜

### 礼盒类
- 百鑫有货 → 百鑫发货
- 百鑫无货 自己店里有货 → 询问用户是否用自己店里的
- 都无货 → 询问是否进货

## 单位换算
- 用户报"X件"时，查 simple_desc 获取每件套数
- 订单数量 = X件 × 每件套数（用套开单）
- "两"是"件"的口语说法
