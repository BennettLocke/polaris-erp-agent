# Changelog

All notable changes to Polaris ERP Agent are documented here.

## [0.1.0] - 2026-06-02

### Added

- Initial public release foundation for the Polaris AI Agent server.
- React admin console for products, inventory, customers, sales orders, workflow orders, settings, image assets, print preview, and AI workbench operations.
- Mini-program API surface for product lists, product detail, customer login, profile, orders, and hot-product analytics.
- Local print task pipeline with the bundled `sjAutoPrint` Windows print agent.
- Deployment guidance for Python, React admin assets, MySQL configuration, systemd, and Docker Compose.

### Release Notes

- This repository is an application/service repository and is versioned through Git tags and GitHub Releases.
- The matching mini-program application is maintained in `BennettLocke/polaris-ai-erp-weapp`.
- The shared UI component library is maintained in `BennettLocke/bennett-locke-ui`.
