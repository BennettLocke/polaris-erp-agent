# 泡袋上传与模板开发交接

这份文档用于新对话快速接上当前进度。

## 当前目标

用户要在北极星智能体里实现“新增泡袋商品”流程：

1. 用户说“开始上传泡袋”。
2. 智能体询问茶类/模板，例如岩茶、红茶、宽版。
3. 用户告诉商品名，例如“九龙窠肉桂”。
4. 用户上传泡袋原始 PNG。
5. 系统自动处理原始图，生成标准泡袋图。
6. 标准图套入模板，生成主图和详情页。
7. 后续再上传到 ERP 创建新商品。

当前先做本地图片处理和模板调参，还没有最终接入 ERP 上传。

## 已确认规则

- 岩茶默认使用当前泡袋模板。
- 商品编号不应该手工从分类取，ERP 里已有自动编号逻辑，从 `SJ0506` 往后递增。
- 商品名由用户提供，例如“虎啸岩肉桂”。
- 分类后续按商品名判断，例如含“肉桂”则走肉桂泡袋分类。
- 模板背景色使用 `#f8f8f8`。
- 固定文案使用：`雅致非凡，尽显格调`。
- 标准泡袋图尺寸：`550 x 1500`。
- 网页模板里的泡袋图片不是 1:1 嵌入，而是按同等长宽比例缩放嵌入模型遮罩。

## 原始图处理

当前新增了 v2 脚本：

`scripts/bag_template/prepare_bag_image_v2.py`

处理逻辑：

1. 通过背景差异/饱和度得到泡袋主体 mask。
2. 先收缩 mask，忽略黑边、毛刺、顶部小开口干扰。
3. 对左右边线做拟合。
4. 通过拟合边线求四角。
5. 顶部轻微内收，减少顶部黑口。
6. 透视拉正到 `550 x 1500`。
7. 如果拟合失败，回退到 raw/minAreaRect 思路。

## 模板调参页

当前可调网页模板：

`scripts/bag_template/bag_manual_adjust.html`

本地同一份也在：

`C:\Users\chuji\Downloads\bag-manual-adjust.html`

模板默认示例图：

`scripts/bag_template/SJ0506-raw-standard.png`

已加功能：

- 图片嵌入统一参数：
  - `统一缩放`
  - `图片 X 偏移`
  - `图片 Y 偏移`
- 用户已确认 `统一缩放 = 1.02` 比较合适。
- 每个模型顶部从左到右约 `3/4` 位置加一个倒三角开口。
- 倒三角深度保持不变，宽度已缩窄到 `模型宽度 * 0.045`。

重要：用户要调的是“泡袋图片在模型遮罩里的统一嵌入参数”，不是分别调每个模型的位置。
